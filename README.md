# ♻️ EcoDetect Abancay - IA de Residuos

Esta es una **PWA (Progressive Web App)** desarrollada con Streamlit y YOLO11 para el reconocimiento automática de residuos sólidos (plastico, papel, vidrio y organico).

## 🚀 Características
- **Detección en tiempo real:** Uso de la cámara del celular.
- **Análisis de fotos:** Carga de imágenes para auditoría rápida.
- **Instalable:** Funciona como una App nativa gracias a su Manifest y Service Worker.

*Proyecto desarrollado para la ciudad de Abancay, Apurímac.*
